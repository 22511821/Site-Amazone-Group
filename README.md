#  Amazone Group – Site Web Officiel

Amazone Group est une entreprise béninoise spécialisée dans le tourisme africain.  
Ce site web présente l’histoire de notre groupe, nos offres, notre restaurant, notre galerie, ainsi qu’un espace de contact professionnel.



## Notre objectif
Créer une plateforme moderne, immersive mais culturelle permettant :
- De promouvoir les destinations touristiques d’Amazone Group.
- De présenter les types de chambres, les délicieux repas et activités.
- De faciliter la prise de contact via un formulaire complet.
- De valoriser le patrimoine culturel du Bénin.


## Technologies utilisées
- **HTML5** – structure du site  
- **CSS3** – design moderne, animations, responsive  
- **Google Fonts & FontAwesome** – typographies et icônes  
- **Git / GitHub** – versionning et collaboration
